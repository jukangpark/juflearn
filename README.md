# Juflearn Project

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## 테스트 코드 실행

```bash
npm run test
```

## test-report 파일 경로

```bash
./coverage/jest-html-reporters-attach/test-report/test-report.html
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
